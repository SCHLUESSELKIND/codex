# Portraits

Hier gehören die Bilddateien von Thomas hin. Solange sie fehlen, zeigen die Vorlagen
einen klar markierten Platzhalter. Es wird bewusst kein Gesicht generiert und kein
Stockbild eingesetzt.

| Datei | Größe | Einsatz |
|---|---|---|
| `tom-thumb.png` | 520 × 720 oder größer, gleiches Verhältnis | Thumbnails |
| `tom-01.png` | frei | allgemeiner Slot |

## Anforderungen

- Freisteller als PNG mit Transparenz, oder sauber auf `#060606` freigestellt
- Kopf im oberen Drittel, Blick in die Kamera oder zur Textzone hin
- Kantenlicht von links, damit die Silhouette gegen Schwarz stehen bleibt
- Schwarzpunkt nicht zufahren, YouTube komprimiert sonst zu Blöcken
- keine Beauty-Retusche, keine künstlichen Reflexe, keine Sci-Fi-Kanten

Zwei bis drei Grundhaltungen reichen: neutral erklärend, konzentriert am Rechner,
direkt in die Kamera. Mehr Varianten schwächen die Wiedererkennung.

Details in `docs/DESIGN_SYSTEM.md`, Abschnitt 8.
